# Inventory Manager

> This README was generated by AI.

Inventory Manager is a C# console application for tracking inventory items, nested child items, transaction history, sorting, validation, and JSON persistence.

## Features

- Add inventory items with unique IDs.
- Look up items by name, ID, or partial name.
- Search recursively through nested child items.
- Remove items and record transactions.
- View or clear transaction history.
- Sort inventory by name, price, or quantity.
- Save inventory data to JSON and load it back later.

## Requirements

- .NET 10 SDK

## Run the project

```bash
dotnet run
```

## Project Structure

- `Program.cs` - application entry point and demo workflow.
- `Classes/` - core application logic, including `InventoryManager`.
- `Models/` - domain models such as `Item` and `Transaction`.
- `Services/` - output, history, repository, and JSON persistence services.
- `SortingTypes/` - sorting strategies for name, price, and quantity.
- `Validations/` - item validation rules.
- `Interfaces/` - contracts used throughout the application.

## Notes

- Item IDs must be unique.
- Item names must not be empty.
- Quantity and price must be non-negative.
